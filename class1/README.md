##3、如何使用github

github公网地址：https://github.com/

git命令：

```
git add . //暂存所有修改 
git commit -m '提交说明' //提交所有暂存文件到本地库
git push //把本地库的变更推送到远程库
git pull //把远程库的变更拉取到本地库

```
